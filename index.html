Tentang Saya

Halo! Saya seorang pengembang web yang memiliki passion di bidang teknologi dan desain. Fokus utama saya adalah menciptakan aplikasi yang modern, responsif, dan user-friendly. Dengan pengalaman di HTML, CSS, JavaScript, dan React, saya terus belajar untuk meningkatkan keahlian saya.

Di waktu luang, saya suka mengeksplorasi teknologi baru dan mengerjakan proyek pribadi untuk memperluas portofolio saya. Saya juga senang berbagi ilmu dengan komunitas pengembang lainnya.
